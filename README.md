# 🛒 Ecommerce Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Ecommerce Sales Dashboard** developed using **Microsoft Power BI**. The dashboard provides insights into sales performance, profitability, customer behavior, product categories, and payment methods using two CSV datasets.

## 🎯 Objectives

- Analyze overall sales and profit.
- Monitor monthly profit trends.
- Compare sales across different states.
- Identify top customers.
- Analyze product categories and sub-categories.
- Understand payment mode preferences.
- Provide interactive filtering using State and Quarter slicers.

## 📊 Business Insights

- Maharashtra has the highest sales amount.
- Clothing category contributes the highest quantity sold.
- COD is the most preferred payment mode.
- Printers generate the highest profit among sub-categories.
- Profit is highest during January and November.
- Top customers contribute significantly to total revenue.

---

## 🚀 How to Use

1. Clone this repository.
2. Open **Ecommerce Sales Dashboard.pbix** in Power BI Desktop.
3. Ensure both CSV files are available in the project folder.
4. Refresh the data if necessary.
5. Explore the dashboard using the Quarter and State slicers.
